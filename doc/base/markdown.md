# Markdown 相关语法
## 一、标题
```
# 这是一级标题
## 这是二级标题
### 这是三级标题
#### 这是四级标题
##### 这是五级标题
###### 这是六级标题
```
+ ### 预览
    + # 这是一级标题
    + # 这是二级标题
    + ## 这是三级标题
    + ### 这是四级标题
    + #### 这是五级标题
    + ##### 这是六级标题

## 二、代码块

用两行 **```** 符号包裹 可以设置代码块高亮 在第一行代码后面跟相关语法例如 `JavaScript`,`Python`等
+ ### 预览
```JavaScript

 }

```
## 三、字体
-  #### 加粗 
    **使用`两个` * 来包裹**
- #### 斜体 
    **使用`一个` * 来包裹**
- #### 斜体加粗 
    **使用`三个` * 来包裹**
- #### 删除文字 
    **使用`两个` ~~ 来包裹**
+ ### 预览
```
**这是加粗的文字**
*这是倾斜的文字*
***这是斜体加粗的文字***
~~这是加删除线的文字~~
```
- **这是加粗的文字**
- *这是倾斜的文字*
- ***这是斜体加粗的文字***
- ~~这是加删除线的文字~~
## 四、引用
#### 在引用的文字前加>即可。引用也可以嵌套，如加两个>>三个>>> n个...
```

>>这是引用的内容
>>>>>>>>>>这是引用的内容
+ 今天天气不错
    >是的这是我说的
```
+ ### 预览
>这是引用的内容
>>这是引用的内容
>>>>>>>>>>这是引用的内容
+ 今天天气不错

## 六、列表
#### 无序列表用 - + * 任何一种都可以

```
1. 无序列表
    - 列表内容
    + 列表内容
    * 列表内容

2. 有序列表

    1. 列表内容
    2. 列表内容
    3. 列表内容

3. 列表嵌套

    + 这是第一层
        + 这是第二层
        + 这是第二层
```
+ ### 预览

## 七、表格
```
这个是左对齐|表头是否居中|这个是右对齐
:---|---|---:
内容|内容|内容
内容|内容|内容

第二行分割表头和内容。

默认左对齐
-右边加：右对齐
-两边加: 居中对齐
-左边加：左对齐
注：原生的语法两边都要用 | 包起来
```
+ ### 预览
这个是左对齐|表头是否居中|这个是右对齐
:---|:---:|---:
内容|内容|内容
内容|内容|内容
## 八、流程图
```
    ```flow
    st=>start: 开始
    op=>operation: My Operation
    cond=>condition: Yes or No?
    e=>end
    st->op->cond
    cond(yes)->e
    cond(no)->op
    ``` 
    注: 有些网站不支持流程图
```
+ ### 预览

```flow
st=>start: 开始
op=>operation: My Operation
cond=>condition: Yes or No?
e=>end
st->op->cond
cond(yes)->e
cond(no)->op
```

## 九、任务列表
 #### 要创建任务列表，请创建一个列表，并以方括号 [ ] 开头。通过添加空格 x 来改变复选框的勾选情况
```
- [x] 吃饭
- [ ] 睡觉
- [ ] 打豆豆
```
+ ### 预览
- [x] 吃饭
- [ ] 睡觉
- [ ] 打豆豆
